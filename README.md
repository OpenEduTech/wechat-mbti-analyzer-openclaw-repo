# wechat-mbti-analyzer-openclaw-repo
这是一个一个面向 OpenClaw 的微信聊天智能分析工作流，帮助用户把分散、非结构化的微信对话，转化为可复用的中文人格与关系分析报告。它支持单聊、群聊、合伙人协作、家庭关系与团队工作模式等多种场景，能够基于本地聊天记录自动完成导出、整理、结构化分析和正式报告初稿生成。与传统的聊天检索或摘要工具不同，这个项目更强调“行为模式洞察”，通过反复出现的表达习惯、决策方式、压力反应和互动节奏，推断人物的 MBTI 倾向及协作关系特征。整个流程支持 Windows 与 macOS，内置 OpenClaw 专用系统提示词、任务模板、跨平台脚本和示例任务，适合直接部署到个人研究、团队管理、合伙人评估和关系观察等场景中。
