# WeChat MBTI Analyzer for OpenClaw 项目介绍

## 项目是什么

`wechat-mbti-analyzer-openclaw-repo` 是一个面向 OpenClaw 的本地工作流仓库，用来把微信聊天记录导出、整理，并进一步生成中文 MBTI 风格的人格分析与协作关系报告。

它不是一个心理测评工具，也不是一个依赖云端服务的 SaaS，而是一套可以在本地运行、可复用、可扩展的聊天分析工作流。

这个仓库的核心目标是：

- 把微信聊天数据整理成结构化分析素材
- 让 OpenClaw 能够稳定复用同一套分析流程
- 输出适合中文语境的正式报告初稿
- 支持个人关系、合伙人关系、团队协作、群聊工作模式等多种场景

## 它解决什么问题

在实际使用里，微信聊天分析通常会卡在三个地方：

- 聊天记录不好稳定导出
- 导出之后缺少统一格式，难以重复分析
- 每次都要重新组织提示词、报告结构和判断维度

这个仓库就是为了解决这三个问题。

它把整个流程拆成了标准化步骤：

1. 通过 `wechat-cli` 导出聊天记录
2. 生成统一的原始文本、元数据和提示词文件
3. 自动区分单聊与群聊
4. 生成正式报告模板
5. 生成中文正式报告初稿

这样你就不需要每次都重新从零搭工作流。

## 适合哪些使用场景

这个仓库适合以下几类任务：

- 分析某一个人的 MBTI 倾向
- 分析你和某个聊天对象的互动模式
- 分析合伙人或创业搭档之间的互补与冲突
- 分析家庭或亲密关系中的沟通风格
- 分析群聊中的团队分工、层级和工作模式
- 对比多个聊天对象，观察同一个人在不同关系中的行为变化

## 项目设计原则

这个仓库不是为了做“绝对正确的人格结论”，而是为了做“有证据支撑的行为风格分析”。

因此它遵循几个基本原则：

- 把 MBTI 当作行为推断，而不是人格诊断
- 优先看重复模式，而不是单条消息
- 工作场景和生活场景分开理解
- 群聊优先分析工作模式，再分析 MBTI
- 输出报告以中文正式表达为主

## 仓库结构说明

仓库的主要结构如下：

- `README.md`
  说明如何在 OpenClaw 中使用这套流程

- `SYSTEM_PROMPT.md`
  OpenClaw 可直接使用的系统提示词

- `TASK_TEMPLATES.md`
  可直接复制的任务模板，覆盖单聊、合伙人、家庭关系、群聊、时间切片等场景

- `SKILL.md`
  面向 Codex / Skill 风格调用的说明文件

- `scripts/`
  核心执行脚本，负责导出、模板生成、初稿生成

- `references/`
  报告结构和表达风格参考材料

- `examples/`
  可直接复制到 OpenClaw 的完整任务示例

## 核心能力

当前这个仓库已经具备以下能力：

- 支持 Windows 和 macOS
- 支持单聊和群聊
- 支持全量导出和时间范围导出
- 支持正式报告模板生成
- 支持中文正式报告初稿生成
- 支持 OpenClaw 直接调用
- 支持整理成 GitHub 仓库进行同步

## 输出结果包含什么

一次完整运行后，通常会生成这些文件：

- `all-messages.txt`
  合并后的聊天文本

- `metadata.json`
  导出时间、消息数、时间范围等元数据

- `mbti-analysis-prompt.md`
  适合继续精修分析的提示词草稿

- `formal-report-template.md`
  正式报告模板

- `formal-report-draft.md`
  中文正式报告初稿

## 为什么适合 OpenClaw

OpenClaw 更适合复用“明确流程 + 明确输入输出 + 明确风格规则”的工作流。

这个仓库正好把这些要素都准备好了：

- 有明确的系统提示词
- 有完整任务模板
- 有跨平台脚本入口
- 有标准化输出文件
- 有可复制的示例任务

所以它不是一堆零散脚本，而是一套适合代理持续复用的工作流包。

## 当前边界

这个仓库也有明确边界：

- 它依赖本地 `wechat-cli`
- 它依赖本机已经能读取微信本地数据
- 它输出的是分析草稿和初稿，不是最终的“绝对结论”
- 它更适合辅助判断和关系理解，不适合替代正式心理测评

## 推荐使用方式

如果你是第一次使用，建议按这个顺序：

1. 先读 `README.md`
2. 再读 `SYSTEM_PROMPT.md`
3. 直接从 `TASK_TEMPLATES.md` 或 `examples/` 复制一个任务
4. 跑通一个单聊分析
5. 再逐步扩展到群聊、关系对比或团队分析

## 一句话总结

这是一个把“微信聊天导出 + MBTI 风格分析 + 中文正式报告生成”打包成可复用 OpenClaw 工作流的仓库，适合做个人关系、合伙协作和团队工作模式分析。
